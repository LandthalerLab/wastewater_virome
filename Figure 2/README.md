1. The kaiju pipelines identifies taxonomy IDs found in the sequencing data. These output files for the Berlin wastewater RNA and DNA data are on the GEO entry (https://www.ncbi.nlm.nih.gov/geo/query/acc.cgi?acc=GSE228220).
